# Gauseva 

Gauseva is a web-based application dedicated to the care, protection, and welfare of cows.  
This project aims to support Gau Seva activities through a simple and user-friendly platform.

---

# Project Overview

The Gauseva project helps manage and support cow welfare initiatives.  
It is built using **Node.js and Express** for the backend, **SQL** for database management, and **HTML, CSS, and JavaScript** for the frontend.

---

#Features

- Informational pages about Gau Seva
- Backend server using Node.js & Express
- SQL database integration
- Simple and responsive UI using HTML & CSS
- JavaScript for frontend interactions
- Secure environment variable handling
